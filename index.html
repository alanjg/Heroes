Heroes

Model animation and rendering for Heroes of Might and Magic 5

Download this project as a .zip file Download this project as a tar.gz file

This is a project to animate and render models from the Heroes of Might and Magic V computer game. Heroes 5 uses a proprietary format for its geometry meshes and the granny 3D format for mesh skeletons and animations, and it stores general model data in an xml format(e.g. texture info, which mesh, skeleton and animations are related, etc). There was an existing tool that converted the mesh file into an SMD file, but that only contained the model geometry and default skeleton, with no animations. I built a converter that could parse the game's data directory, extract all models and their related skeletons and animations, and prepare it for rendering. Since the game uses a special mesh format, the default Granny 3D animation code won't work correctly, so I reverse engineered and re-implemented the Granny 3D spline animation routines. The skeleton system is a hierarchical system of animated transforms with various weights applied to each transform, and with position, rotation, and scale animation tracks using B-Splines. The renderer uses a vertex shader to apply the different transforms.
